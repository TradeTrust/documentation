## Sign up for a Stability Protocol's API Key

Follow [Stability's guide](https://docs.stabilityprotocol.com/developers/register_for_api_key) to sign up for a Stability Protocol's API Key

## Sign up for Infura's API Key

:::tip
Refer to Infura official documentation [here](https://docs.metamask.io/developer-tools/dashboard/get-started/create-api/)
:::

### Step 1: Navigate to Infura

Navigate to [infura.io](https://app.infura.io/).

And Sign up for a free account or log in to your existing account. 

Infura will automatically generate the `My First Key` API key for new users.
Click the API Key.

<img
  style={{ margin: "auto", marginBottom: "2rem" }}
  src={"/docs/topics/advanced/additional-networks-to-metamask-guide/infura-home.png"}
  alt="Click API Key"
/>

### Step 2: Enable Service / Network

Enable endpoints from list of networks and click `Save Changes`.

<img
  style={{ margin: "auto", marginBottom: "2rem" }}
  src={"/docs/topics/advanced/additional-networks-to-metamask-guide/infura-select-network.png"}
  alt="Enable endpoints"
/>

### Step 3: Copy API Key

Click the icon ( highlighted with red border ), to copy the API Key.

<img
  style={{ margin: "auto", marginBottom: "2rem" }}
  src={"/docs/topics/advanced/additional-networks-to-metamask-guide/infura-copy-key.png"}
  alt="Copy API Key"
/>

Now that you have the **API Key**, you can add it into your MetaMask. You can refer to the [guide](#fill-in-the-network-configuration-for-the-new-network-as-required) above.
