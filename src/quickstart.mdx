# TradeTrust Quick Start Guide

### Quick Start (only pdf bundle issuance):

1. Install Metamask, procure Ethers (1 eth from [faucet](https://faucet.metamask.io/) will be enough for a long time)
1. Install the [Open-Attestation CLI tool](https://openattestation.com/docs/component/open-attestation-cli)
1. Deploy a [document store](../deploying-store)
1. [Create DNS-TXT Record](../dns-txt) for document store
1. [Use the document creator to generate trade documents](../document-creator)


### Creating Custom Templates:

1. Complete Quick Start steps above first
1. Fork and customize [GitHub - TradeTrust/document-renderer](https://github.com/TradeTrust/document-renderer)
1. Add `$template` field the config for the document creator 
1. Issue using [the document creator](../document-creator)
